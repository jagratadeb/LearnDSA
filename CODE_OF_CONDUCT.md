# Code of Conduct

## Our Pledge
This repository is shared openly for learning and reference.  
Although contributions are not accepted, I expect anyone who engages with the project (through forks, issues, or discussions) to maintain a respectful and professional tone.

## Our Standards
Examples of positive behavior include:
- Using the code responsibly and giving proper credit
- Respecting the intent of the repository (no unsolicited pull requests)
- Being courteous in any discussions or references to this project

Examples of unacceptable behavior include:
- Harassment, trolling, or offensive comments
- Misrepresentation of the project or its author
- Using the project in ways that promote discrimination or harm

## Scope
This Code of Conduct applies to all spaces where the repository is referenced, including forks, discussions, and external usage.

## Enforcement
Violations may result in reporting to GitHub or removal of interactions (e.g., issues or comments).  
The repository owner reserves the right to enforce these standards to maintain a safe and professional environment.

## Attribution
This Code of Conduct is inspired by the [Contributor Covenant](https://www.contributor-covenant.org), version 2.1, adapted for repositories not open to public contributions.
